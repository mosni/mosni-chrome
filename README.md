# mosnicat

The shared visual chrome and design system for the mosni stack — a drop-in of CSS and native
web components. No framework, no build step for consumers.

## Use it

Add one tag to your page's `<head>`:

```html
<script src="https://ui.mosni.dev/mosnicat.js"></script>
```

That loads the stylesheet and registers every `<mosni-*>` component. Nothing else to wire up.

## Docs

Live examples and the full component/class reference: **https://ui.mosni.dev**
